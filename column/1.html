AWSの日本への投資

「日本に2兆円超」──AWSの投資が意味するものとは

2024年1月、Amazon Web Services（AWS）は、日本のクラウドインフラに2兆2,600億円の追加投資を行うと発表しました。東京リージョンと大阪リージョンの拡張を中心とした取り組みで、再生可能エネルギーや雇用の拡大にもつながるとしています。

「クラウドって、技術者の話でしょ？」
「正直、自分には関係ないと思う」

そう思う方もいるかもしれません。でもこの投資によって、私たちが使う行政サービスや日常的に触れるITの裏側が静かに変わっていくのです。その変化は目には見えませんが、社会の基盤を少しずつ、しかし確実に更新していく力を持っています。

AWSの投資は、サーバーだけの話ではない

AWSが表明した投資は、クラウドの構築にとどまりません。

  • 東京・大阪リージョンのデータセンターの設備拡張
  • 施設運用を支える電力・通信インフラの整備
  • 再生可能エネルギーによる電力供給の推進
  • 地方における人材育成と雇用創出

AWSはこの投資により、日本のGDPに約5.57兆円の経済効果と、年間30,500人の雇用創出が見込まれるとしています。

出典：AWS公式発表 (2024)

日本におけるAWSの存在感

日本では、クラウドサービス（PaaS/IaaS）を導入している企業の60%以上がAWSを利用しており、その存在感は非常に大きいと言えます。 総務省の白書によれば、PaaS導入企業のうち60.0%がAWSを使用しており、Azure（48.2%）、GCP（28.8%）を上回る結果となっています。

出典：総務省『令和6年 情報通信白書』、MM総研「国内クラウドサービス需要動向調査」（2022年6月）

なお、これらの数値は「複数サービスの併用」を含む利用率であり、市場全体のシェアとは異なります。ただしこの結果からも、日本企業におけるAWSの導入率は高水準であることが読み取れます。

政府も選んだクラウド基盤

2021年、日本政府は「ガバメントクラウド（政府共通クラウド）」の整備を進める方針を発表しました。

2025年時点では以下の5つのクラウドベンダーが採用されています。

  • Amazon Web Services（AWS）
  • Google Cloud Platform（GCP）
  • Microsoft Azure
  • Oracle Cloud Infrastructure（OCI）
  • さくらインターネット

これは、民間だけでなく政府の基幹システムもクラウド化し、その一部をAWSが担っていることを意味します。

出典：AWS公式（2023年）

出典：デジタル庁 ガバメントクラウド

余談になりますが、デジタル庁はnoteを書いているのですね。 ガバメントクラウドにおけるモダン化の意味と定義というnoteを見つけたのですが、 クラウド設計のベストプラクティスがわかりやすく書かれています。覗いてみるとおもしろいです。

データ管理と「クラウド主権」の考え方

クラウドの活用が進む中で、「どこにデータが保管され、誰がアクセス権限を持つのか」が世界的に注目されています。

ヨーロッパでは、クラウドサービスに対する主権の確保を目的に、独立クラウド「AWS European Sovereign Cloud」を構築中です。最初のリージョンはドイツに設置され、EU規制や公共機関の要求に応える設計がなされています。

日本でも、こうしたデータの透明性や運用責任についての意識はすでに高まっており、その一環としてさくらインターネットがガバメントクラウドの対象事業者として選定されるなど、具体的な取り組みが進められています。

出典：AWS Europe Digital Sovereignty

次回：日本のクラウドはどう挑むのか？

AWSの日本での投資は、日本のITインフラをより強固にし、経済や行政にも大きな影響をもたらすものです。
しかし一方で、「海外のクラウドベンダーに依存してよいのか」という観点も、同時に考える必要があります。

次回は、さくらインターネット、NTT、富士通といった国産クラウド事業者の取り組みと課題に焦点を当てます。
技術的な挑戦、政府対応、そしてそれぞれの独自戦略に注目しながら、日本のクラウドの未来を一緒に考えていきましょう。
