※このサイトについて

このサイトは社内勉強会 for AWS用の学習支援サイトです。内容の正確性に努めていますが、誤りや仕様変更の可能性もあるため、詳細は公式ドキュメントをご確認ください。
学習目的（非商用）であれば個人・企業問わず利用可能ですが、勉強会は社内限定です。

1. 学べること

AWS（Amazon Web Services） は、Amazonが提供するクラウドサービスで、サーバーやデータベースなど様々なITリソースを提供しています。この勉強会では、VPCやIAMなどの基礎に加え、可用性・スケーラビリティといったクラウド設計の考え方、主要サービスの使い方を解説とハンズオンで学びます。初心者や、AWSに少し触れたことがある方向けです。

3. 費用について

  • 主に無料枠の範囲で学習できる設計（全セッションの合計費用は数百円に納めます）
  • 各回の終了後に使用したリソースの削除確認
  • AWS Budgetsで意図しない課金時にメール通知を設定
  • 二段階認証（MFA）で不正利用を防止

4. サイトの構成

  • セッション：各回の教材・操作手順・資料
  • ナレッジベース：AWSサービスや関連技術の補足資料
  • コラム：クラウド関連の情報と私見

  • PC(Windows,Mac,Linux)推奨
  • Chrome/Firefox(拡張機能無効化)
  • AWSアカウント作成用メールアドレス、電話番号、クレジットカード

6. よくある質問（FAQ）

Q：途中参加でも大丈夫ですか？
A：大歓迎です！セッションの内容はサイトで確認できるため、途中参加や、お休みしても大丈夫です！

Q：質問や連絡はどこでできますか？
A：質問板(※実装予定)やTeams?で随時対応します！

Q：実務でも役立ちますか？
A：はい、実務でも役立ちます！ クラウドの活用は今後さらに広がると考えられ、特にAWSは多くの現場で使われています。 フロントエンド・バックエンド・インフラをまたいで働く機会も増えており、学んでおいて損はありません。

7. 最後に

このサイトはAWSのS3とCloudFrontを使ってホスティングしています。質問版にはAPI Gateway、Lambda、RDSの使用を検討中です。
もし時間に余裕があれば、自作したウェブサイトやウェブアプリを学習したAWS上にデプロイしてみてください。知識やスキルがより身につくだけでなく、それを証明するものとしても役に立つと思います。
勉強会は気軽に参加でき、リラックスして学べる場になるよう心がけます。興味のある方はぜひ一緒に学んでいきましょう！気になることがあれば遠慮なくご質問ください！
